How to hack: Write some random stuff to create awesome hacker-code.

Features:

1 + Enter = Access Granted
2 + Enter = Access Denied
3 + Enter = Upload Bar then Access Granted
Shift = clear pop-ups
Ctrl = toggle Matrix
