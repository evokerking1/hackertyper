How to hack: Write some random stuff to create awesome hacker-code.

Features:

1 + Enter = Access Granted <br />
2 + Enter = Access Denied <br />
3 + Enter = Upload Bar then Access Granted <br />
Shift = clear pop-ups <br />
Ctrl = toggle Matrix <br />
